# Configure Mem0 OSS Components

<Info>
  **Prerequisites**
  - Node.js 18 or higher with `npm` or `pnpm` available
  - Running vector database (e.g., Qdrant, Postgres + pgvector) or access credentials for a managed store
  - API keys for your chosen LLM, embedder, and reranker providers
</Info>

<Tip>
  Start from the <Link href="/open-source/node-quickstart">Node.js quickstart</Link> if you still need the base install and first memory example.
</Tip>

## Install the SDK

```bash
npm install mem0ai
```

## Define your configuration

Pass a configuration object to `new Memory(config)` to override any default component.

```ts
import { Memory } from "mem0ai/oss";

const memory = new Memory({
  vectorStore: {
    provider: "qdrant",
    config: {
      host: "localhost",
      port: 6333,
      collectionName: "my-memories",
    },
  },
  llm: {
    provider: "openai",
    config: {
      apiKey: process.env.OPENAI_API_KEY,
      model: "gpt-4o-mini",
      temperature: 0.1,
    },
  },
  embedder: {
    provider: "openai",
    config: {
      apiKey: process.env.OPENAI_API_KEY,
      model: "text-embedding-3-small",
    },
  },
  reranker: {
    provider: "cohere",
    config: {
      apiKey: process.env.COHERE_API_KEY,
      model: "rerank-english-v3.0",
    },
  },
});
```

Store secrets as environment variables:

```bash
export OPENAI_API_KEY="..."
export COHERE_API_KEY="..."
```

## Validate the setup

```ts
await memory.add(
  [{ role: "user", content: "My favorite cafe in Tokyo is Fuglen." }],
  { userId: "alex" }
);

const results = await memory.search("favorite cafe", { userId: "alex" });
console.log(results);
```

## Tune component settings

<AccordionGroup>
  <Accordion title="Vector store collections">
    Name collections explicitly in production (`collectionName`) to isolate tenants and enable per-tenant retention policies.
  </Accordion>
  <Accordion title="LLM extraction temperature">
    Keep extraction temperatures <=0.2 so extracted memories stay deterministic. Raise it only when you see missing facts.
  </Accordion>
  <Accordion title="Reranker depth">
    Limit `topK` to 10-20 results; sending more adds latency without meaningful improvement.
  </Accordion>
  <Accordion title="Disable history tracking">
    Set `disableHistory: true` in the config to skip SQLite history writes for higher-throughput pipelines.
  </Accordion>
</AccordionGroup>

<Warning>
  Mixing managed and self-hosted components? Make sure every outbound provider call happens through a secure network path. Managed rerankers often require outbound internet even if your vector store is on-prem.
</Warning>

## Quick recovery

- **Qdrant connection errors** - confirm port `6333` is exposed and the collection name matches.
- **Empty search results** - verify the embedder model name; a dimension mismatch silently returns zero results.
- **`Unknown provider`** - update the SDK (`npm install mem0ai@latest`) to load the latest provider registry.
