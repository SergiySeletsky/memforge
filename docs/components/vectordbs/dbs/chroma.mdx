[Chroma](https://www.trychroma.com/) is an AI-native open-source vector database that simplifies building LLM apps with a focus on simplicity and speed. It supports both local deployment and cloud hosting through ChromaDB Cloud.

### Usage

#### Local Installation

```ts
import { Memory } from "mem0ai/oss";

const memory = new Memory({
  vectorStore: {
    provider: "chroma",
    config: {
      collectionName: "test",
      path: "db",
    },
  },
});

const messages = [
  { role: "user", content: "I'm planning to watch a movie tonight. Any recommendations?" },
  { role: "assistant", content: "How about thriller movies? They can be quite engaging." },
  { role: "user", content: "I'm not a big fan of thriller movies but I love sci-fi movies." },
  { role: "assistant", content: "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future." },
];

await memory.add(messages, { userId: "alice", metadata: { category: "movies" } });
```

#### ChromaDB Cloud

```ts
const memory = new Memory({
  vectorStore: {
    provider: "chroma",
    config: {
      collectionName: "test",
      apiKey: process.env.CHROMA_API_KEY,
      tenant: process.env.CHROMA_TENANT_ID,
    },
  },
});
```

#### Remote Chroma Server

```ts
const memory = new Memory({
  vectorStore: {
    provider: "chroma",
    config: {
      collectionName: "test",
      host: "localhost",
      port: 8000,
    },
  },
});
```

### Config

Here are the parameters available for configuring Chroma:

| Parameter | Description | Default Value |
| --- | --- | --- |
| `collectionName` | The name of the collection | `mem0` |
| `path` | Path for the local Chroma database | `db` |
| `host` | Host where the Chroma server is running | `undefined` |
| `port` | Port where the Chroma server is running | `undefined` |
| `apiKey` | ChromaDB Cloud API key (for cloud usage) | `undefined` |
| `tenant` | ChromaDB Cloud tenant ID (for cloud usage) | `undefined` |
