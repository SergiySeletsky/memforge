Cohere provides enterprise-grade reranking models with excellent multilingual support and production-ready performance.

## Models

Cohere offers several reranking models:

- **`rerank-english-v3.0`**: Latest English reranker with best performance
- **`rerank-multilingual-v3.0`**: Multilingual support for global applications
- **`rerank-english-v2.0`**: Previous generation English reranker

## Installation

```bash
npm install cohere-ai
```

## Environment Variables

```bash
export COHERE_API_KEY="your-api-key"
```

## Configuration

```ts
import { Memory } from "mem0ai/oss";

const memory = new Memory({
  vectorStore: {
    provider: "chroma",
    config: { collectionName: "my_memories", path: "./chroma_db" },
  },
  llm: {
    provider: "openai",
    config: { model: "gpt-4o-mini" },
  },
  reranker: {
    provider: "cohere",
    config: {
      model: "rerank-english-v3.0",
      apiKey: process.env.COHERE_API_KEY,
      topK: 5,
      returnDocuments: false,
    },
  },
});
```

## Usage Example

```ts
import { Memory } from "mem0ai/oss";

const memory = new Memory({
  vectorStore: { provider: "chroma", config: { collectionName: "my_memories", path: "./chroma_db" } },
  llm: { provider: "openai", config: { model: "gpt-4o-mini" } },
  reranker: {
    provider: "cohere",
    config: { model: "rerank-english-v3.0", apiKey: process.env.COHERE_API_KEY, topK: 3 },
  },
});

// Add memories
await memory.add([
  { role: "user", content: "I work as a data scientist at Microsoft" },
  { role: "user", content: "I specialize in machine learning and NLP" },
  { role: "user", content: "I enjoy playing tennis on weekends" },
], { userId: "bob" });

// Search with reranking
const results = await memory.search("What is the user's profession?", { userId: "bob" });
results.results.forEach((r: any) => {
  console.log(`Memory: ${r.memory}`);
  console.log(`Score: ${r.score.toFixed(3)}`);
});
```

## Multilingual Support

For multilingual applications, use the multilingual model:

```ts
const memory = new Memory({
  reranker: {
    provider: "cohere",
    config: {
      model: "rerank-multilingual-v3.0",
      topK: 5,
    },
  },
});
```

## Configuration Parameters

| Parameter            | Description                      | Type      | Default                 |
| -------------------- | -------------------------------- | --------- | ----------------------- |
| `model`              | Cohere rerank model to use       | `string`  | `"rerank-english-v3.0"` |
| `apiKey`             | Cohere API key                   | `string`  | `undefined`             |
| `topK`               | Maximum documents to return      | `number`  | `undefined`             |
| `returnDocuments`    | Whether to return document texts | `boolean` | `false`                 |
| `maxChunksPerDoc`    | Maximum chunks per document      | `number`  | `undefined`             |
